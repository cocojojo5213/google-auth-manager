# Google 授权应用管理器

🔐 **提取并备份您的 Google 授权应用，提供快捷链接设置备用登录方式。**

## ✨ 功能特点

- 📋 从 Google 权限页面**提取已授权应用**
- 🔗 收录 **200+ 服务**的登录、设置、安全页面链接
- 🌐 **中英双语**界面
- 💾 **导出备份**为 JSON 或文本
- ➕ **手动添加**遗漏的应用

## 🚀 为什么需要这个？

如果 Google 账号丢失，那些只用"通过 Google 登录"的服务可能就无法访问了。

这个工具帮助您：
1. 查看所有与 Google 账号关联的服务
2. 快速跳转到各服务的安全设置
3. 设置邮箱+密码作为备用登录方式

## 📖 使用方法

1. **打开 Google 权限页面**
   - [myaccount.google.com/connections](https://myaccount.google.com/connections)

2. **复制页面内容**
   - 按 `Ctrl+A`（全选），再按 `Ctrl+C`（复制）

3. **粘贴并解析**
   - 粘贴到文本框，点击"解析"

4. **点击链接**
   - 跳转到各服务的设置/安全页面
